## Generating extensions using elements

Suppose we have two fields $ K/F$ with $ \alpha \in K$. We can know that the collection of subfields of $ K$ containing both $ F$ and $ \alpha$ is nonempty (as $ K$ is in this collection). We can then intersect all such fields and arrive at a _unique minimal_ subfield of $ K$ containing $ F$ and $ \alpha$. Then notice that we can do this for any collection of elements of $ F$. We put this reasoning into its own definition. 

{{% MathEnv "defn" %}}
Let $ K$ be an extension of the field $ F$ and let $ \alpha, \beta, \ldots \in K$ be a collection of elements of $ K$. Then the smallest subfield of $ K$ containing both $ F$ and the elements $ \alpha,\beta,\ldots$, denoted $ F(\alpha,\beta,\ldots)$ is called the field {{% tdf "generated by $ \alpha,\beta,\ldots$ over $ F$" %}}. 
{{% /MathEnv %}}

It turns out that we'll end up studying a lot of fields where we are extending by only one element. Indeed, we give these extensions their own name.

{{% MathEnv "defn" %}}
If the field $ K$ is generated by a single element $ \alpha$ over $ F$, $ K = F(\alpha)$, then $ K$ is said to be a {{% tdf "simple extension of $ F$" %}}  and the element of $ \alpha$ is called a {{% tdf "primitive element" %}} for the extension.
{{% /MathEnv %}}

{{% MathEnv "rem" %}}
We also sometimes read $ F(\alpha)$ as "$ F$ adjoin $ \alpha$". 
{{% /MathEnv %}}


## Quotienting by Polynomials and Adjoining Roots

I mentioned previously that one of my complex analysis professors said that
$$
    \mathbb{C} \cong \frac{ \mathbb{R}[t] }{(t^2+1)}.
$$
Inspired by our previous discussion on generating fields by single elements and recalling that $ \mathbb{C}$ has dimension $ 2$ as an $ \mathbb{R}$-vector space, we can see that $ \mathbb{C} \cong \mathbb{R}(i)$. Since isomorphic-ness is transitive, we now have that 
$$
    \frac{ \mathbb{R}[t] }{ (t^2+1) } \cong \mathbb{R}(i).
$$
Now to blow your (read: my) mind a lot more, notice that $ \mathbb{R}(i) \cong \mathbb{R}(-i)$, giving us that 
$$
   \mathbb{R}(-i) \cong \frac{ \mathbb{R}[t] }{ (t^2+1) } \cong \mathbb{R}(i).
$$      
This helps illustrate that when we quotient by a (irreducible) polynomial, we are actually just adjoining a root (in fact, any root of our irreducible) to our field! Indeed, we can make this observation rigorous. :)

{{% MathEnv "thm" %}}
Let $ F$ be a field and let $ p(t) \in F[t]$ be an irreducible polynomial. Suppose $ K$ is an extension of $ F$ containing a root $ \alpha$ of $ p(t)$: $ p(\alpha)=0$. Then: 
$$
    F(\alpha) \cong \frac{ F[t] }{ (p(t)) }.
$$
{{% /MathEnv %}}

{{% Proof %}}
We prove this statement using the First Isomorphism Theorem. Let $ \phi \colon F[t] \to F(\alpha)$ defined by $ F[t] \ni f(t) \mapsto f(\alpha) \in F(\alpha)$. (Essentially we consider an evaluation map.) By hypothesis $ p(\alpha) = 0$, so $ p(t) \in \operatorname{ker}(\phi)$ so $ (p(t)) \subseteq \operatorname{ker}(\phi)$ and since $ p(t)$ is irreducible and $ F[t]$ is a PID, $ (p(t))$ is maximal, so $ \operatorname{ker}(\phi) = (p(t))$. Then by the First Isomorphism Theorem, there is a map $ \psi \colon F[t]/((p(t)) \to \operatorname{im}(\psi) $ that witnesses this isomorphism. Now, since $ \operatorname{im}(\psi)$ is an ideal of $ F(\alpha)$, $ F(\alpha)$ has no nontrivial ideals (as it is a field), and $ 1 \in \operatorname{im}(\psi)$, it necessarily follows that $ \operatorname{im}(\psi) = F(\alpha)$. Thus 
$$
   \frac{ F[t] }{ (p(t)) } \cong F(\alpha)
$$
as desired.
{{% /Proof %}}
{{% MathEnv "rem" %}}
Notice that we didn't specify which root of $ p(t)$ was being adjoined. Thus we have shown that quotienting by an irreducible polynomial is the same as adjoining all of its roots. This will be of great importance soon enough.
{{% /MathEnv %}}

## Algebraic extensions

Before hearing about algebraic numbers, you likely heard of "transcendental numbers" like $ \pi$ and $ e$. You might have also heard that these numbers are "not being the root of any polynomial over $ \mathbb{Q}$". Indeed, not only does this generalize to any field, but we are often more interested in its dual: an algebraic number/element. As you would expect, an algebraic number _is_ the root of a polynomial.

{{% MathEnv "defn" %}}
If $ K/F$, $ \alpha \in K$ is said to be {{% tdf "algebraic over $ F$" %}} if $ \alpha$ is a root of some nonzero polynomial $ f(t) \in F[t]$. If $ \alpha$ is not algebraic (i.e., not the root of any nonzero polynomial with coefficients in $ F$) then $ \alpha$ is a said to be {{% tdf "transcendental over $ F$" %}}.
{{% /MathEnv %}}

In light of this definition of algebraic elements, we now define an _algebraic extension_. 

{{% MathEnv "defn" %}}
An extension $ K/F$ is said to be {{% tdf "algebraic" %}} if every element of $ K$ is algebraic over $ F$.            
{{% /MathEnv %}}

{{% MathEnv "rem" %}}
Notice that if you are algebraic over the base field, you are algebraic over any extension. (This is because you are the root of a polynomial with coefficients in $ F$ and extensions of $ F$ necessarily contain $ F$.)
{{% /MathEnv %}}

## Minimal polynomials

Recall from Theorem 4 earlier in this post that if $ F$ is a field, $ p(t) \in F[t]$ is irreducible, and if $ K$ is an extension of $ F$ containing a root $ \alpha$ of $ p(t)$, then $ F[t]/(p(t)) \cong F(\alpha)$. In particular, if we translate this theorem in our newly learned language of algebraicity we get that: adjoining an algebraic element to a field is isomorphic to quotienting by a polynomial that witnesses its algebraicity. Let's study this "witness polynomial" for a bit. 

This $ p(t)$ is said to be irreducible with $ \alpha$ as a root. What can we say about such an object? 
- Since its irreducible over $ F[t]$, we can assume that $ p(t)$ is monic (i.e., its leading coefficient is $ 1$) as otherwise we could multiply it by a constant without changing the roots (hence reducing the polynomial).  
- If we consider our map (to which we applied the First Isomorphism Theorem in the proof of Theorem 4) $ \phi \colon F[t] \to F(\alpha)$, if $ f(t) \in F[t]$ has $ \alpha$ as a root, then $ f(t) \in \operatorname{ker}(\phi) = (p(t))$. Hence $ p(t) \mid f(t)$. 

We call such a $ p(t)$ a _minimal polynomial_ as we can't factor it anymore and any other polynomial that has $ \alpha$ as a root is divisible by $ p(t)$. We state this result now (though we have only sketched the proof and will not give it in full detail):

{{% MathEnv "prop" %}}
Let $ \alpha$ be algebraic over a field $ F$. Then there is a unique monic irreducible polynomial $ m_F(\alpha)(t) \in F[t]$ which has $ \alpha$ as a root. Moreover, a polynomial $ f(t) \in F[t]$ has $ \alpha$ as a root if and only if $ m_F(\alpha) \mid f$ in $ F[t]$.
{{% /MathEnv %}}

We also get the following corollary.

{{% MathEnv "coro" %}}
If $ L/F$ is an extension of fields and $ \alpha$ is algebraic over both $ F$ and $ L$, then $ m_L(\alpha) \mid m_F(\alpha)$.
{{% /MathEnv %}}
{{% Proof %}}
Noticing that $ F \subseteq L$, it becomes clear that $ m_F(\alpha) \in L[t]$ and hence, by definition of the minimal polynomial, $ m_L(\alpha) \mid m_F(\alpha)$ as desired.
{{% /Proof %}}

Just for completeness we state the definition of the minimal polynomial.
{{% MathEnv "defn" %}}
The polynomial $ m_F(\alpha)$ in Proposition 9 is called he {{% tdf "minimal polynomial for $ \alpha$ over $ F$" %}} . The degree of $ m_F(\alpha)$ is called the {{% tdf "degree of $ \alpha$" %}}.
{{% /MathEnv %}}


## Algebraic extensions continued

Now that we've developed the machinery known as minimal polynomials, we can make quite the interesting (yet hopefully unsurprising) claim about the degree of a simple algebraic extension!

{{% MathEnv "thm" %}}
Let $ \alpha$ be algebraic over the field $ F$ and let $ F(\alpha)$ be the field generated by $ \alpha$ over $ F$. Then 
$$
    F(\alpha) \cong \frac{ F([t]) }{ (m_F(\alpha)) }.
$$
In particular, we have that 
$$
    [F(\alpha) : F] = \operatorname{deg}(m_F(\alpha)) = \operatorname{deg}(\alpha).
$$
{{% /MathEnv %}}
{{% Proof %}}
This follows immediately from Theorem 4 and Proposition 9.
{{% /Proof %}}

Now we have the following. 

{{% MathEnv "prop" %}}
The element $ \alpha$ is algebraic over a field $ F$ if and only if the simple extension $ F(\alpha) / F$ is finite. 
{{% /MathEnv %}}

{{% Proof %}}
We first prove the first part of the claim. ($\Rightarrow$) Suppose $ \alpha$ is algebraic over $ F$. Then by Theorem 12, $ [F(\alpha) : F] = n < \infty$ (as an element is algebraic if and only if it has finite degree). ($\Leftarrow$) Suppose $ F(\alpha) /F$ is finite. Then $ \operatorname{deg}(\alpha) = [F(\alpha) : F] < \infty$ and $ \alpha$ is algebraic. 
{{% /Proof %}}

This gives rise to two important corollaries.

{{% MathEnv "coro" %}}
More precisely, if $ \alpha$ is an element of an extension of degree $ n$ over $ F$, then $ \alpha$ satisfies a polynomial of degree at most $ n$ over $ F$ and if $ \alpha$ satisfies a polynomial of degree $ n$ over $ F$ then $ [F(\alpha):F] \leq n$.
{{% /MathEnv %}}

{{% MathEnv "coro" %}}
If the extension $ K/F$ is finite, then it is algebraic.
{{% /MathEnv %}}

## The tower property

If $ L/K/F$, what is the degree of $ L/F$ if $ L$ is algebraic as an extension of $ K$ and $ K$ is algebraic as an extension of $ F$? Well, since we know degrees are just the dimension of its vector space, we can do a basis-counting argument. We refer to the answer of this question as the _tower property_ and we state and prove it now.

{{% MathEnv "thm" %}}
Let $ F \subseteq K \subseteq L$ be fields. Then 
$$
    [L:F] = [L:K][K:F].
$$
{{% /MathEnv %}}
{{% Proof %}}
We first prove the finite case. Suppose $ [L:K] = m$ and $ [K:F]=n$ are finite. Let $\alpha_1,\ldots,\alpha_m$ be a $ K$-basis of $ L$. We then for all $ \ell \in L$, there exists $ a_1,\ldots,a_m \in K$ such that 
$$
    \ell = a_1 \alpha_1 + \cdots + a_m \alpha_m.
$$
Then  letting $ \beta_1,\ldots,\beta_n$ to be a $ F$-basis of $ K$, we get that each $ a_i$ may be written as 
$$
    a_i = b_{i,1} \beta_1 + \cdots + b_{i,n} \beta_n.
$$      
Hence we get that 
$$
    \ell =\sum_{i=1}^{m} a_i \alpha_i = \sum_{i=1}^{m}\sum_{j=1}^{n} b_{i,j} \beta_j \alpha_i.
$$
Since $ b_{i,j} \in F$ for all $ i,j$, we have shown that $ \left \\{ \alpha_i \beta_j : 1 \leq i \leq m \text{ and } 1 \leq j \leq n \right \\} $ spans $ L$.{{%nl%}}{{%nl%}}

Suppose we had 
$$
    \sum_{i=1}^{m}\sum_{j=1}^{n} b_{i,j} \beta_j \alpha_i = 0.
$$
Then we may use $ a_i =\sum_{j=1}^{n}b_{i,j} \beta_j$ to get that 
$$
   \sum_{i=1}^{m} a_i \alpha_i = 0.
$$
Since $ a_i \in K$ and $ \left \\{ \alpha_1,\ldots,\alpha_m \right \\} $ is a basis of $ L$, 
$$
    a_i = 0 \quad \text{ for all } 1 \leq i \leq m.
$$
Thus 
$$
   \sum_{j=1}^{n} b_{i,j} \beta_j = 0
$$
for all $ 1 \leq i \leq m$, and by similar reasoning, each $ b_{i,j} = 0$ as $ \left \\{ \beta_1,\ldots,\beta_n \right \\} $ is an $ F$-basis of $ K$. Thus 
$$
    [L:F] = mn = [L:K][K:F].
$$

In the infinite cases, we must only observe that the product of two finite values is finite, the product of an infinite and a finite value is infinite, and the product of two infinite values is infinite. 
{{% /Proof %}}

We then get a corollary that reminds me very deeply of Lagrange's Theorem from group theory.

{{% MathEnv "coro" %}}
Suppose $ L/F$ is a finite extension and let $ K$ be an subfield of $ L$ containing $ F$, $ F \subseteq K \subseteq L$. The $ [K:F]$ divides $ [L:F]$.
{{% /MathEnv %}}

## Finitely-generated extensions

We first define what _finitely generated_ means. (It's not very surprising.)

{{% MathEnv "defn" %}}
An extension $ K/F$ is {{% tdf "finitely generated" %}} if there are elements $ \alpha_1,\ldots,\alpha_k$ in $ K$ such that $ K = F(\alpha_1,\ldots,\alpha_k)$.
{{% /MathEnv %}}

Intuitively, repeated extending my algebraic elements should result in an algebraic (read: finite degree) extension. But how do we prove this? To me, it's not especially obvious that we can do this when given $ \alpha_1,\ldots,\alpha_k$ as that's pretty difficult to check linear independence before (I imagine), but it turns out that we can just adjoin the algebraic elements one at a time! Indeed:

{{% MathEnv "lem" %}}
$ F(\alpha,\beta) = (F(\alpha))(\beta)$, i.e., the field generated over $ F$ by $ \alpha$ and $ \beta$ is the field generated by $ \beta$ over the field $ F(\alpha)$ generated by $ \alpha$. 
{{% /MathEnv %}}
{{% Proof %}}
Clearly $ \alpha, \beta \in (F(\alpha))(\beta)$ so $ (F(\alpha))(\beta) \supseteq F(\alpha,\beta)$ by minimality. Similarly, $ \alpha \in F(\alpha)$ so $ F(\alpha) \subseteq F(\alpha,\beta)$. Hence $ (F(\alpha))(\beta) \subseteq (F(\alpha,\beta))(\beta) = F(\alpha,\beta)$. Thus $ (F(\alpha))(\beta) = F(\alpha,\beta)$. 
{{% /Proof %}}

Now using the tower property with this lemma, we can compute the degree of these repeated extensions pretty easily!

{{% MathEnv "thm" %}}
If $ F $ is a field and $ \alpha_1,\ldots,\alpha_k $ are algebraic over $ F $ and $ F_n = F(\alpha_1,\ldots,\alpha_n) $, $ F_0 = F $, and $ K  = F_k $, we have that 
$$
    [K:F] = [F_k : F_0] = [F_k : F_{k-1}] [F_{k-1}: F_{k-2}] \cdots [F_1 : F_0]
$$
{{% /MathEnv %}}
{{% Proof %}}
This is an immediate application of writing $ F_{n+1} $ as $ F_n(\alpha_{n+1}) $, noting that $ F_k/F_{k-1}/\cdots/F_1/F_0 $, and using the lemma above.
{{% /Proof %}}

We also get this nice corollary.
{{% MathEnv "coro" %}}

If $ F $ is a field and $ \alpha_1,\ldots,\alpha_k $ are algebraic of degrees $ n_1,\ldots,n_k $, respectively, and $ K = F(\alpha_1,\ldots,\alpha_k) $ we have that 
$$
    [K:F] \leq n_1 \cdots n_{k-1} n_k
$$
{{% /MathEnv %}}
